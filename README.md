# Social-Media-Analytics-Dashboard
📊 Social Media Analytics Dashboard

📌 Project Overview

This project focuses on analyzing social media performance data and transforming it into actionable business insights using **Python**, **MySQL**, and **Power BI**.

The dashboard provides a comprehensive view of content performance, engagement behavior, posting patterns, and audience growth to support data-driven social media strategies.

---

🎯 Business Objectives

The dashboard answers key business questions such as:

* What is the best media type based on engagement performance?
* What is the optimal posting hour?
* Which content categories generate the highest impressions?
* What day of the week delivers the strongest engagement?
* Which account type performs best?
* How do different media types impact follower growth?
* How effective are calls-to-action across content categories?

📂 Project Workflow

1. Data Cleaning & Preparation

* Handled missing values
* Removed duplicate records
* Created derived metrics
* Validated data consistency

2. Feature Engineering

Created additional analytical features including:

* Total Engagements
* Engagement Rate by Reach
* Engagement Rate by Impressions
* Day of Week
* Posting Hour
* Viral Post Indicator

3. Exploratory Data Analysis (EDA)

Performed:

* Distribution Analysis
* Outlier Detection
* Boxplot Analysis
* Content Performance Comparison
* Engagement Pattern Analysis

4. SQL Analysis

Used MySQL to answer business questions through:

* Aggregations
* Window Functions
* Ranking Analysis
* Median Calculations
* Performance Comparisons

5. Dashboard Development

Built an interactive Power BI dashboard featuring:

* KPI Cards
* Trend Analysis
* Category Performance
* Media Type Comparison
* Posting Time Analysis
* Interactive Slicers
